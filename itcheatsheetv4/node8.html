Linux Mint 

Ebooks-
Install Calibre from website to get latest version:
sudo -v && wget -nv -O- https://download.calibre-ebook.com/linux-installer.sh | sudo sh /dev/stdin
Install DeDRM plugin.
apt install python3-pip
pip3 install epy-reader
epy /path/to/epub_book

Latex-
Install TexStudio from software manager GUI.
apt install texlive
