Backup Raspberry Pi SD Card

Backup Raspberry pi sd card: 
lsblk

mount -t msdos /dev/da2s2 /mnt/sd

dd if=/dev/da2 if=/home/phils/rasp_backup.img status=progress

Restore Raspberry pi sd card:

dd if=/home/phils/rasp_backup.img if=/dev/da2
